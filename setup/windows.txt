# Windows Setup

1. **Download and install Node.js from the official site:**
   https://nodejs.org/

2. **Verify that Node.js and npm are installed:**
   Open Command Prompt and run:
   node -v
   npm -v

3. **Navigate to the cloned project directory:**
   cd path\to\your\react-intro-ws

4. **Install project dependencies:**
   npm install
   npm install tailwindcss
   npm install aos

5. **Start the Development Server:**
   npm run dev

6. **You can now view the project at `http://localhost:3000`.**
