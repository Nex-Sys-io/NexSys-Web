# macOS Setup

1. **Install Homebrew (if you don't have it installed):**
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

2. **Install Node.js using Homebrew:**
   brew install node

3. **Check if Node.js and npm are installed:**
   node -v
   npm -v

4. **Navigate to the cloned project directory:**
   cd path/to/your/react-intro-ws

5. **Install project dependencies:**
   npm install
   npm install tailwindcss
   npm install aos

6. **Start the Development Server:**
   npm run dev

7. **You can now view the project at `http://localhost:3000`.**
